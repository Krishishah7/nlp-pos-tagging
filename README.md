# NLP POS Tagging

This project demonstrates Part-of-Speech (POS) tagging in Natural Language Processing (NLP) using Python and the NLTK library.

The goal of this project is to identify the grammatical role of each word in a sentence, such as noun, verb, adjective, or adverb.
